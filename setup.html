Instrucciones

  1. Ve a supabase.com y crea una cuenta o inicia sesión
  2. Crea un nuevo proyecto en Supabase
  3. Ve a Settings > API
  4. Copia el Project URL y pégalo en el campo "Supabase URL"
  5. Copia el anon public key y pégalo en el campo "Supabase Anon Key"
  6. En el SQL Editor de Supabase, ejecuta el contenido de supabase-setup.sql
