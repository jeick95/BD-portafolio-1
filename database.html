Configuración Paso a Paso

Sigue estos pasos para conectar tu aplicación a Supabase

Paso 1: Obtener Credenciales de Supabase

Ve a tu proyecto en Supabase y copia las credenciales desde Settings > API.

// URL del Proyecto(ejemplo)
https://dcsnngfnfxgnfafbnzfgn.supabase.co
// Clave pública (anon key)(ejemplo)
vrh8b7vebnei8snbdnkrjn.rgjrhj4b4j3hf8bsyfbsu7vshvsudhf

Paso 2: Crear archivo config.js

Crea el archivo js/config.js con las credenciales de tu proyecto.

const SUPABASE_URL = 'https://dcsnngfnfxgnfafbnzfgn.supabase.co';
const SUPABASE_ANON_KEY = 'vrh8b7vebnei8snbdnkrjn.rgjrhj4b4j3hf8bsyfbsu7vshvsudhf';
const supabase = Supabase.createClient(SUPABASE_URL, SUPABASE_ANON_KEY);
const SUPABASE_URL = 'https://dcsnngfnfxgnfafbnzfgn.supabase.co';
const SUPABASE_ANON_KEY = 'vrh8b7vebnei8snbdnkrjn.rgjrhj4b4j3hf8bsyfbsu7vshvsudhf';

const supabase = Supabase.createClient(SUPABASE_URL, SUPABASE_ANON_KEY);

Paso 3: Incluir librería en tu HTML

Añade el script de Supabase en tu archivo HTML antes de tus scripts.

<!-- Supabase -->
<script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2"></script>
<!-- Tu config -->
<script src="js/config.js"></script>

Paso 5: Políticas de Seguridad (RLS)

Configura Row Level Security para proteger tus datos.

-- Habilitar RLS
ALTER TABLE profiles ENABLE ROW LEVEL SECURITY;
-- Permitir lectura pública
CREATE POLICY "Lectura publica"
ON profiles FOR SELECT USING (true);

Características de Supabase

Potentes herramientas que ofrece Supabase

Tiempo Real

Escucha cambios en tu base de datos en tiempo real con suscripciones.

Autenticación

Sistema completo de auth con email, OAuth, y más.

Storage

Almacena archivos e imágenes en la nube.

APIs Automáticas

Genera REST y GraphQL APIs automáticamente.

Información Importante

Seguridad

Nunca expongas tu clave de servicio (service_role) en el código del cliente. Usa la clave anónima (anon) para operaciones públicas y configura RLS para proteger tus datos.

Uso en Java

Para conectar desde una aplicación Java, usa la API REST de Supabase o bibliotecas como Unirest, OkHttp, o el cliente oficial de Supabase para Java. La conexión funciona igual que cualquier API REST.
